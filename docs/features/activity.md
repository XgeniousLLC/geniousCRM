# Activity Feed

The Activity Feed is an automatic log of everything that happens in your CRM. You never need to manually write what you did — Mini CRM logs it for you.

## What Gets Logged

| Action | When |
|--------|------|
| **Created** | Contact, lead, deal, or company is created |
| **Updated** | Contact or lead is edited |
| **Stage Changed** | A deal moves to a new pipeline stage |
| **Status Changed** | A lead's status is updated |
| **Note Added** | A note is added to a contact or lead |
| **Note Deleted** | A note is removed |
| **Converted** | A lead is converted to a contact |
| **Activity Logged** | A manual deal log entry (email, call, etc.) is added |

## Global Activity Feed

Go to **Activity Feed** in the sidebar (visible to Admin and Manager roles). This shows all recent activity across the entire CRM — who did what, when, on which record.

Each entry shows:
- **Actor** — the user who performed the action
- **Action** — what they did
- **Entity** — the contact/lead/deal they acted on (clickable link)
- **Time** — relative timestamp (e.g. "2 hours ago")

## Per-Entity Activity

On a contact, lead, deal, or company detail page, scroll to the **Activity** section to see only actions related to that specific record.

## Filtering

On the global feed you can filter by:
- Entity type (Contacts, Leads, Deals)
- Date range

## Who Can See the Feed

| Role | Access |
|------|--------|
| Admin | Full global feed |
| Manager | Full global feed |
| Sales User | Only activity on records assigned to them |
