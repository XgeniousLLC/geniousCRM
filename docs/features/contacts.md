# Contacts

Contacts are the people at the centre of your CRM. Every contact can be tagged, linked to a company, given notes, and associated with leads, deals, and tasks.

## The Contacts List

Go to **Contacts** in the sidebar. The table shows:

| Column | Description |
|--------|-------------|
| Name | Full name (first + last), links to the contact detail page |
| Email | Primary email address |
| Phone | Phone number |
| Company | Linked company name |
| Tags | Colour-coded tag badges |
| Actions | Edit, Delete |

### Searching

Type in the search box at the top to filter by name, email, or company. Results update as you type.

### Filtering by Tag

Click a tag badge in the filter bar to show only contacts with that tag.

## Creating a Contact

Click **New Contact** (top right). Fill in the form:

| Field | Required | Notes |
|-------|----------|-------|
| First Name | Yes | |
| Last Name | Yes | |
| Email | No | Must be unique if provided |
| Phone | No | |
| Company (text) | No | Free-text company name |
| Company (linked) | No | Link to a Company record |
| Tags | No | Select from existing tags or create new |

Click **Save**. The contact appears in the list immediately.

## Editing a Contact

Click the **Edit** (pencil) icon on any row. The same form opens pre-filled. Make your changes and click **Save**.

## Deleting a Contact

Click the **Delete** (trash) icon. Confirm the dialog. The contact is **soft-deleted** — it moves to the [Contacts Trash](/features/trash) where it can be restored within 30 days.

## Contact Notes

Open a contact's detail page by clicking their name. The **Notes** drawer lets you:

- **Add a note** — type in the text box and click **Add Note**
- **Delete a note** — click the × on any note you created

Notes are timestamped and show the author's name.

## Linked Tasks

The contact detail page also shows all tasks linked to this contact. You can create new tasks from here — see [Tasks](/features/tasks).

## CSV Import

To import contacts in bulk, click **Import CSV** on the Contacts list page. See the [CSV Import](/features/csv-import) guide for column mapping and rules.

## Tags

Tags are shared across contacts and leads. To manage the tag list, simply type a new tag name in any contact's tag field — it will be created automatically.

Common tags to start with: `VIP`, `Hot Lead`, `Cold`, `Partner`, `Enterprise`, `SMB`.
