# Features

Every feature in Mini CRM lives inside its own Laravel module under `Modules/`. Select a feature below for a detailed guide.

| Feature | Module | Description |
|---------|--------|-------------|
| [Dashboard](dashboard) | Core | Summary stats and quick links |
| [Authentication](authentication) | Auth | Login, register, 2FA, password reset |
| [Contacts](contacts) | Contact | CRM contacts with tags, notes, import |
| [Leads](leads) | Lead | Prospect pipeline with follow-up tracking |
| [Deals](deals) | Deal | Sales pipeline with Kanban board |
| [Tasks](tasks) | Task | Polymorphic tasks and notes |
| [Activities](activities) | Activity | Auto-logged audit feed |
| [Companies](companies) | Company | Organisation profiles |
| [Reports](reports) | Report | Charts, stats, CSV export |
| [Notifications](notifications) | Notification | In-app bell + email alerts |
| [CSV Import](csv-import) | Contact/Lead | Bulk import from spreadsheets |
| [Trash](trash) | Core | Restore or permanently delete records |
| [Users & Roles](users-roles) | User | User management and RBAC |
| [Settings](settings) | Core | Logo, favicon, meta tags |
| [Security](security) | Auth | 2FA, rate limiting, session management |
| [Deal Enhancements](deal-enhancements) | Deal | Products, close alerts, win probability |
| [Global Search](search) | Core | Cross-entity instant search |
| [Bulk Operations](bulk-operations) | Contact/Lead | Select many, act once |
| [Dark Mode](dark-mode) | Core | Light/dark theme toggle |
| [Keyboard Shortcuts](keyboard-shortcuts) | Core | Hotkeys for power users |
| [Onboarding](onboarding) | Core | First-time setup wizard |
| [Timezone Support](timezone) | Core | Per-user IANA timezone |
