# Dashboard

The Dashboard is the first page you see after logging in. It gives a real-time summary of your CRM health.

## Stat Cards

Six summary cards sit at the top of the page:

| Card | Description |
|------|-------------|
| **Total Contacts** | All active contacts in the CRM |
| **Total Leads** | Active leads (excludes converted and lost) |
| **Open Deals** | Deals in stages other than Won or Lost |
| **Open Tasks** | Tasks with status Pending or In Progress |
| **Follow-ups Due** | Leads where the follow-up date is today or past |
| **Closing This Week** | Deals whose expected close date is within 7 days |

Click any card to jump to the relevant list with the matching filter pre-applied.

## Recent Activity Feed

Below the stat cards, the last 10 activity entries are shown — contacts created, deals moved, notes added, etc. Click **View all** to open the full [Activity Feed](/features/activity).

## Role Differences

| Role | Behaviour |
|------|-----------|
| Admin | All stat cards, all activity |
| Manager | All stat cards, all activity |
| Sales User | Counts scoped to their own assigned records |

## Navigation Tips

- Use the **left sidebar** to move between modules.
- On mobile, tap ☰ to open the sidebar drawer.
- Press `?` anywhere to see all keyboard shortcuts.
